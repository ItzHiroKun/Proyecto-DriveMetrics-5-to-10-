# Proyecto DriveMetrics 5-to-10

Este repositorio es utilizado para ir subiendo y actualizando los archivos de mi proyecto junto a mi compañero de universidad. Aquí iremos creando distintos microservicios que conectaremos entre sí.

## Herramientas utilizadas

- **XAMPP** para utilizar:
  - **Apache** como servidor web.
  - **Tomcat** como contenedor de aplicaciones Java.
  - **phpMyAdmin** para la gestión de bases de datos MySQL.
- **Extensiones adicionales** para facilitar el desarrollo y la comunicación entre los microservicios.
